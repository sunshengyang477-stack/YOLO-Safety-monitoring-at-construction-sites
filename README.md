# 基于 YOLO 的施工现场安全智能监测系统

## 1. 项目简介
本项目是一款专门针对建筑施工现场开发的智能安全监管系统。系统利用深度学习算法 **YOLOv8**，通过实时视频流或图像识别技术，自动监测工人是否佩戴**安全帽**、穿戴**反光背心**及其他防护用品。系统旨在通过 AI 自动化监管降低施工现场的安全隐患，提升行业本质安全水平。

## 2. 核心功能
* **多目标实时检测**：支持对施工现场的人员、安全帽、反光背心等多个目标进行同步识别。
* **多源输入支持**：系统可接入本地图片、MP4/AVI 视频文件以及 USB/网络摄像头实时流。
* **智能交互界面**：基于 PyQt5 开发的可视化 GUI，支持双屏对比展示（原始画面 vs 检测标注画面）。
* **违规取证快照**：集成“一键保存”与“自动截图”功能，发现违规行为时可实时捕获带标注的证据图片。
* **动态参数调节**：允许用户在界面实时调整置信度（Conf）和交并比（IoU）阈值，以适应不同光照环境。

## 3. 技术栈
* **开发语言**：Python 3.9+
* **深度学习框架**：PyTorch
* **核心算法**：Ultralytics YOLOv8m (Medium)
* **图形界面**：PyQt5
* **图像处理**：OpenCV-Python
* **硬件加速**：NVIDIA CUDA & cuDNN

## 4. 算法表现
根据中期测试数据（基于 `results.csv`）：
* **模型选型**：YOLOv8m
* **训练状态**：已完成 200/500 轮（Epochs）训练。
* **精度指标**：在复杂工地背景下，mAP50（平均精度）已达到 **0.506** 并呈现稳步上升趋势，展现了良好的特征提取能力。



## 5. 系统架构
项目采用多线程异步架构（Multithreading），确保高负载检测任务不阻塞 UI 响应：
1.  **主线程 (Main Thread)**：负责 PyQt5 界面渲染与用户指令分发。
2.  **监测线程 (Detection Thread)**：独立负责视频帧获取、YOLO 模型推理及图像后处理。
3.  **数据持久化层**：负责训练日志记录与 `results/` 文件夹下的截图管理。
